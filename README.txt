퀸잇 리뷰 작성기

1. start.ps1을 PowerShell로 실행합니다.
2. 브라우저에서 http://127.0.0.1:4173 을 엽니다.
3. 퀸잇 상품 ID를 쉼표 또는 줄바꿈으로 여러 개 입력할 수 있습니다.
4. 성별, 연령대, 말투, 리뷰 길이, 추가 명령을 설정한 뒤 '상품 불러오기'를 누릅니다.
   - 리뷰 길이는 1줄부터 5줄까지 선택할 수 있습니다.
   - 말투에서 '채팅'을 선택하면 ㅎㅎㅎ, ㅋㅋㅋ, ~ 같은 인터넷 채팅 표현이 자연스럽게 포함됩니다.
5. 상품별 5개 리뷰가 표에 표시됩니다.
   - 컬러가 2개면 대표 컬러 3개, 다른 컬러 2개로 배분합니다.
   - 컬러가 3개 이상이면 대표 컬러 3개, 다른 컬러를 1개씩 배분합니다.
   - 각 리뷰 행의 옵션 컬러를 변경하면 해당 리뷰만 자동으로 새로 작성됩니다.
   - 옵션 코드와 리뷰 사이의 '말투 구분자'에서 리뷰마다 서로 다른 말투를 선택할 수 있습니다.
   - 말투를 변경하면 해당 리뷰 한 개만 새로운 내용으로 다시 작성됩니다.
   - 각 리뷰 행에서 1~5줄 길이를 따로 선택할 수 있으며, 변경 시 해당 리뷰만 다시 작성됩니다.
   - 명령어 입력란에는 기본값 없이 사용 예시만 표시됩니다.
6. 상품 전체 또는 리뷰 한 개만 새로고침할 수 있습니다.
7. '엑셀 다운로드'를 누르면 필수 항목이 입력된 XLSX 파일이 저장됩니다.

상품명·이미지·컬러·사이즈는 퀸잇 상세페이지에서 확인합니다.
기존 옵션 자료에 없는 상품은 학습한 옵션 코드 패턴으로 코드를 추론하므로,
실제 판매자 어드민 값과 다를 수 있습니다. 다운로드 전에 확인해 주세요.

OpenAI API 키를 저장하면 Windows DPAPI로 암호화되며 현재 Windows 계정에서만 사용할 수 있습니다.
OpenAI 요청 한도를 모두 사용한 경우에는 오류로 중단되지 않고 같은 줄 수와 말투를 적용한 기본 생성 방식으로 자동 전환됩니다.
한도 초과 시 웹페이지 상단에 예상 리셋까지 남은 시간이 실시간 카운트다운으로 표시됩니다.
배포용 ZIP에는 저장된 API 키 파일이 포함되지 않습니다.
